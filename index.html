HTML

HTML est l'abbrévation d'HyperText Markup Langage. C'est un langage qui sert à créer des sites web statiques, c'est à dire le fond et la structure du site.
C'est un langage qui est interprété par les navigateurs comme du texte et qui fonctionne avec des centaines de balises.
Les balises permettent d'indiquer la structure et les caractéristiques de ce que l'on veut implémenter dans notre site web.
HTML est donc un langage de balises qui se veut sémantique. Il est super important d'utiliser les balises qui ont du sens pour ce que l'on écrit.

Les différentes balises

1. Les balises de base pour commencer un document html

Voici les différentes balises essentielles pour commencer un document HTML

  • Pour commencer tout document html : !DOCTYPE html
  • La balise qui permet d'indiquer un document html : html
  • La balise d'infos sur le document : head
  • La balise du "corps" d'html : body
  • Les balises de titre : h1, h2, h3, h4, h5 et h6
  • La balise pour encadrer un paragraphe : p

Les deux grands types de balises : en bloc et en ligne
